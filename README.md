# RoadMap Builder

Web app para crear y gestionar roadmaps personales con estética japonesa.

## Funcionalidades

- Crear y editar roadmaps con fases e ítems
- Checklists con progreso por fase
- Guardado automático en localStorage
- Descargar roadmap como archivo `.roadmap`
- Importar un archivo `.roadmap` para seguir editando

## Stack

- React + Vite
- CSS personalizado (estética japonesa)
- Bootstrap
- localStorage (sin backend)
